## Iniciando el proyecto en local

Primero, para correr el proyecto de manera local, debes iniciar en una terminal el siguiente comando

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) y veras el resultado en tu navegador predeterminado

Puedes empezar modificando el `app/page.js`. y se vera reflejado los cambios en el navegador
